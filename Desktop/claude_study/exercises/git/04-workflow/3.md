## Задание

Пройди полный цикл разработки мини-проекта:

1. Создай репозиторий на GitHub с README.

2. Настрой:
   - Branch protection rule на main (require PR)
   - Labels: feature, bug, docs, enhancement

3. Создай 3 Issues:
   - #1: "Добавить HTML структуру" (feature)
   - #2: "Добавить стили" (feature)
   - #3: "Написать README" (docs)

4. Для каждого Issue:
   a) Создай ветку (feature/html-structure, feature/styles, docs/readme)
   b) Сделай изменения + коммит с конвенцией (feat:, docs:)
   c) Запуши ветку
   d) Создай PR с описанием и "Closes #N"
   e) Слей PR

5. Посмотри: все Issues закрыты? Все ветки слиты?

6. Финальная проверка:
   ```bash
   git log --oneline --graph
   git branch -a
   ```

7. Напиши чеклист: что делать при работе с GitHub в команде?
