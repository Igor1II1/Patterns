# Задание 1: Pull Request на GitHub
**Уровень:** 🟢 Лёгкий
**Тема:** PR, code review, merge strategies
**Ссылка:** notes/git/04-github-workflow.md — "Pull Request"

---

## Задание

1. Создай ветку feature/about-page.
2. Добавь файл about.html, закоммить, запуши.
3. Создай Pull Request на GitHub:
   - Title: "Добавить страницу О нас"
   - Description: что добавлено, зачем
   - Reviewer (если работаешь в команде)

4. Объясни:
   - Что такое Pull Request?
   - Зачем нужен code review?
   - Какие стратегии merge существуют? (Merge commit, Squash, Rebase)
   - Когда какую использовать?

5. Слей PR через GitHub (кнопка "Merge pull request").
6. Подтяни изменения локально: `git pull origin main`
7. Удали ветку: `git branch -d feature/about-page`
