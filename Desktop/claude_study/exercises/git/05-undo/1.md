## Задание

1. Отменить незакоммиченные изменения в файле:
   ```bash
   # Изменил файл, хочу вернуть как было
   git restore index.html
   ```

2. Убрать файл из staging (git add → хочу отменить add):
   ```bash
   git restore --staged index.html
   ```

3. git reset — отмена коммитов:
   ```bash
   # Отменить последний коммит (сохранить изменения в working directory)
   git reset --soft HEAD~1

   # Отменить последний коммит (сохранить изменения unstaged)
   git reset --mixed HEAD~1  # или просто git reset HEAD~1

   # Отменить последний коммит (удалить изменения!)
   git reset --hard HEAD~1  # ОПАСНО!
   ```

4. Практика:
   - Сделай коммит → отмени через reset --soft → проверь status
   - Сделай коммит → отмени через reset --mixed → проверь status
   - Объясни разницу между soft, mixed, hard
