## Задание

Реши каждую "аварийную" ситуацию:

1. **Ситуация:** Закоммитил секрет (.env с паролем) и запушил.
   **Решение:** ? (git revert? Новый коммит? Смена пароля обязательна!)

2. **Ситуация:** Сделал 3 коммита, хочу объединить в один.
   **Решение:** git reset --soft HEAD~3 → git commit

3. **Ситуация:** Случайно удалил ветку с важным кодом.
   **Решение:** git reflog → найти hash → git switch -c branch hash

4. **Ситуация:** Работаю в wrong ветке, хочу перенести изменения.
   **Решение:** git stash → git switch right-branch → git stash pop

5. **Ситуация:** git reset --hard удалил нужные коммиты.
   **Решение:** git reflog → git reset --hard hash

6. **reflog** — история всех действий:
   ```bash
   git reflog
   # Показывает ВСЕ действия, даже после reset --hard
   # Можно восстановить практически что угодно
   ```

7. Напиши "аптечку Git" — какую команду использовать в какой ситуации:
   | Ситуация | Команда |
   |----------|---------|
   | Отменить изменения в файле | git restore |
   | Убрать из staging | git restore --staged |
   | Отменить коммит (локальный) | git reset |
   | Отменить коммит (запушенный) | git revert |
   | Спрятать изменения | git stash |
   | Восстановить удалённое | git reflog |
