## Задание

Имитируй работу над проектом:

1. Создай проект с main веткой и initial commit.

2. Создай 3 feature-ветки и работай в них:
   - `feature/navigation` — добавь навигацию
   - `feature/content` — добавь контент
   - `feature/styles` — добавь стили

3. Слей каждую ветку в main по очереди:
   - `git switch main && git merge feature/navigation`
   - Повтори для остальных

4. Создай конфликт: измени один файл в двух ветках.
   Разреши его.

5. Посмотри граф: `git log --oneline --graph --all`

6. Удали слитые ветки: `git branch -d feature/navigation`

7. Напиши шпаргалку:
   - `git branch` — список веток
   - `git switch -c name` — создать и переключиться
   - `git merge name` — слить ветку
   - `git branch -d name` — удалить ветку
