# Задание 1: Создание и переключение веток
**Уровень:** 🟢 Лёгкий
**Тема:** branch, checkout/switch, merge
**Ссылка:** notes/git/02-branches.md — "Ветки"

---

## Задание

1. Создай ветку и переключись:
   ```bash
   git branch feature/header
   git switch feature/header
   # или одной командой:
   git switch -c feature/header
   ```

2. Внеси изменения в новой ветке (добавь header в HTML).
   Закоммить.

3. Переключись на main: `git switch main`
   Убедись что header исчез (он только в feature/header).

4. Слей ветку в main:
   ```bash
   git switch main
   git merge feature/header
   ```

5. Удали ветку: `git branch -d feature/header`

6. Повтори для feature/footer.

7. Объясни:
   - Что такое ветка в Git? (указатель на коммит)
   - Зачем нужны ветки? (параллельная работа)
   - Что делает merge? (объединяет изменения)
