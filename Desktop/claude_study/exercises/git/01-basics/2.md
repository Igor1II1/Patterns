## Задание

1. Измени `index.html` (добавь текст).
   Посмотри изменения: `git diff`
   Что показывает? (+ зелёные строки, - красные)

2. Добавь в staging: `git add index.html`
   Теперь `git diff` пустой — почему?
   Используй `git diff --staged` чтобы увидеть staged изменения.

3. Создай файл `.gitignore` с содержимым:
   ```
   node_modules/
   .env
   .env.local
   .next/
   dist/
   *.log
   .DS_Store
   ```

4. Создай файл `test.log` и папку `node_modules/`.
   Выполни `git status` — появились ли они? (Нет, gitignore работает!)

5. Закоммить `.gitignore`.

6. Объясни: зачем .gitignore? Что будет если закоммитить node_modules?
