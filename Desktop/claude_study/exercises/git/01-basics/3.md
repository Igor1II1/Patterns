## Задание

Выполни полный цикл работы:

1. Создай новый проект с 3 файлами:
   - index.html
   - style.css
   - script.js

2. Добавь .gitignore (node_modules, .env, etc.)

3. Сделай initial commit со всеми файлами.

4. Внеси изменения в каждый файл:
   - `git diff` — посмотри все изменения
   - `git add style.css` — добавь только один файл
   - `git status` — один staged, два modified
   - `git commit -m "Обновить стили"`
   - `git add .` — добавь остальные
   - `git commit -m "Обновить HTML и JS"`

5. Посмотри историю: `git log --oneline --graph`

6. Какие команды ты использовал? Напиши шпаргалку:
   - init, status, add, commit, diff, log
   - Для каждой: что делает + когда использовать
