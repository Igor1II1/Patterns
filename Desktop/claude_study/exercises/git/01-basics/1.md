## Задание

Выполни в терминале:

1. Создай папку `git-practice` и инициализируй репозиторий:
   ```bash
   mkdir git-practice && cd git-practice
   git init
   ```

2. Создай файл `index.html` с базовым содержимым.
   Проверь статус: `git status`
   Что показывает? (Untracked files)

3. Добавь файл в staging: `git add index.html`
   Снова `git status` — что изменилось?

4. Сделай первый коммит: `git commit -m "Добавить index.html"`

5. Создай `style.css`, добавь и закоммить.

6. Посмотри историю: `git log`, `git log --oneline`

7. Объясни в комментарии три области Git:
   - Working Directory
   - Staging Area (Index)
   - Repository
