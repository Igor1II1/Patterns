**Уровень:** 🟢 Лёгкий

## Задание

1. Создай репозиторий на GitHub (через github.com).

2. Подключи remote:
   ```bash
   git remote add origin https://github.com/username/repo.git
   git remote -v
   ```

3. Запуши код:
   ```bash
   git push -u origin main
   ```
   Что делает `-u`? (устанавливает upstream)

4. Склонируй репозиторий в другую папку:
   ```bash
   git clone https://github.com/username/repo.git repo-clone
   ```

5. В клоне: внеси изменения, закоммить, запуши.

6. В оригинале: подтяни изменения:
   ```bash
   git pull
   ```

7. Объясни:
   - Что такое remote? (ссылка на удалённый репозиторий)
   - Чем push отличается от pull?
   - Чем clone отличается от fork?
