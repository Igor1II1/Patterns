## Задание

Имитируй совместную работу (два клона одного репозитория):

1. Создай репозиторий на GitHub и склонируй дважды:
   ```bash
   git clone ... developer-1
   git clone ... developer-2
   ```

2. Developer-1: создай ветку, сделай изменения, запуши:
   ```bash
   cd developer-1
   git switch -c feature/auth
   # изменения + commit
   git push -u origin feature/auth
   ```

3. Developer-2: подтяни ветку и добавь свои изменения:
   ```bash
   cd developer-2
   git fetch origin
   git switch feature/auth
   # изменения + commit + push
   ```

4. Developer-1: подтяни изменения Developer-2:
   ```bash
   git pull origin feature/auth
   ```

5. Создай конфликт и разреши его.

6. Слей ветку в main и запуши.

7. Объясни весь workflow: fork → clone → branch → commit → push → PR → merge
