## Слабые места для проработки
- Falsy/truthy ловушки (0 и "" — falsy, но валидные значения)
- return vs console.log
- if vs else if (независимые проверки vs одна ветка)
- Логические выражения && || ??
- Имена свойств объектов (user.email vs user.emailVerified)
- Знаки сравнения в циклах (>= vs <=, < vs >)
- Параметр функции (n) vs переменная цикла (i)
- push() vs = для массивов
- continue пропускает то, что В условии
