## Задание

1. Напиши middleware проверки владельца доски:
   ```
   PUT /api/boards/:boardId — только owner
   DELETE /api/boards/:boardId — только owner
   POST /api/boards/:boardId/columns — только owner
   ```

2. Напиши функцию:
   ```typescript
   async function verifyBoardOwner(boardId: number, userId: string): Promise<boolean>
   ```

3. Опиши правила доступа для TaskBoard:
   - Кто может видеть доску?
   - Кто может редактировать?
   - Кто может удалить?
   - Кто может перемещать задачи?

4. Напиши middleware, который:
   - Проверяет авторизацию
   - Проверяет владельца ресурса
   - Возвращает 401/403 при нарушении

5. Объясни: почему проверка прав на бэкенде важнее проверки на фронтенде?
