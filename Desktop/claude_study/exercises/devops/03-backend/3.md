## Задание

Спроектируй и напиши полный backend для TaskBoard:

1. **Все эндпоинты** (с кодом):
   - Boards: GET list, GET by id, POST, PUT, DELETE
   - Columns: GET by board, POST, PUT (rename/reorder), DELETE
   - Tasks: GET by column, POST, PUT (edit/move), DELETE

2. **Zod-схемы** для каждого эндпоинта.

3. **Проверки доступа**:
   - Авторизация на всех мутациях
   - Проверка владельца доски
   - Участники доски (если добавлять shared boards)

4. **Специальные эндпоинты**:
   - PUT /api/tasks/:id/move — перемещение задачи между колонками
     (обновить columnId + пересчитать order)
   - PUT /api/columns/reorder — изменить порядок колонок

5. **Обработка ошибок**: единый формат для всех эндпоинтов.

6. **Транзакции**: перемещение задачи — атомарная операция.
