## Задание

1. Напиши шаблон API Route Handler, который включает:
   - Проверку авторизации
   - Парсинг и валидацию body через Zod
   - try/catch с обработкой ошибок Prisma
   - Правильные HTTP-коды ответов
   - Логирование

2. Примени шаблон для эндпоинта POST /api/tasks:
   - Проверить авторизацию
   - Валидировать: title (min 1), columnId (number), description? (optional)
   - Создать задачу через Prisma
   - Вернуть 201 + задачу

3. Напиши helper-функции:
   - `withAuth(handler)` — обёртка для проверки сессии
   - `validateBody(schema, body)` — валидация через Zod
   - `handleError(error)` — универсальный обработчик ошибок
