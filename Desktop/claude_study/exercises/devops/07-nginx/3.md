## Задание

1. Добавь Nginx в docker-compose.yml:
   - image: nginx:alpine
   - volumes: nginx.conf, certs
   - ports: 80, 443
   - depends_on: app

2. Напиши nginx.conf для Docker:
   - upstream backend { server app:3000; }
   - Вместо localhost используй имена сервисов

3. Напиши конфигурацию балансировки между 3 копиями:
   - upstream backend { server app1:3000; server app2:3000; server app3:3000; }
   - docker compose up --scale app=3

4. Объясни алгоритмы балансировки:
   - Round-robin (по умолчанию)
   - least_conn
   - ip_hash
