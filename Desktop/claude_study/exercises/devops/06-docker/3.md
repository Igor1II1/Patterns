## Задание

   - Stage 2 (runner): только production файлы
   - Используй `output: 'standalone'` в next.config.js

2. Сравни размеры:
   - Обычный Dockerfile: ~1 GB
   - Multi-stage: ~100-200 MB
   Почему такая разница?

3. Добавь безопасность:
   - Создай пользователя nextjs (не root)
   - USER nextjs

4. Объясни:
   - Что делает COPY --from=builder?
   - Зачем output: 'standalone'?
   - Почему не запускать от root?
