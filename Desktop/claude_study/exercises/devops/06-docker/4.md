## Задание

Создай полную Docker-инфраструктуру для TaskBoard:

1. **Multi-stage Dockerfile** для Next.js (оптимизированный)
2. **.dockerignore** (полный список исключений)
3. **docker-compose.yml** (production):
   - app (Next.js)
   - db (PostgreSQL + healthcheck + volume)
   - nginx (reverse proxy, порт 80)
4. **docker-compose.dev.yml** (разработка):
   - Только БД
5. **nginx.conf**: проксирование на app:3000
6. **Команды** для каждого сценария:
   - Разработка: поднять только БД
   - Тестирование: поднять всё
   - Production: собрать и запустить

7. Объясни workflow:
   - Разработка: docker compose -f docker-compose.dev.yml up + npm run dev
   - Production: docker compose up --build -d
