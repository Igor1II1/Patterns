## Задание

1. Напиши Dockerfile для Next.js приложения:
   - FROM node:20-alpine
   - WORKDIR, COPY, RUN npm ci, COPY, RUN build
   - EXPOSE 3000, CMD

2. Напиши .dockerignore (что исключить).

3. Напиши команды:
   - Сборка образа: `docker build -t taskboard .`
   - Запуск: `docker run -p 3000:3000 taskboard`
   - Запуск с env: `docker run -p 3000:3000 -e DATABASE_URL=... taskboard`

4. Объясни каждую инструкцию Dockerfile.

5. Почему COPY package.json отдельно от COPY . ?
   (Кеширование слоёв Docker)
