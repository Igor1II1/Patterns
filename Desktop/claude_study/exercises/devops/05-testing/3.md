## Задание

Разработай стратегию тестирования для TaskBoard:

1. **Unit-тесты** (перечисли что тестировать):
   - Утилиты (formatDate, slugify, validateEmail)
   - Zod-схемы валидации
   - Helper-функции (calculateOrder, canEditTask)

2. **Компонентные тесты** (перечисли компоненты):
   - TaskCard, CreateTaskForm, BoardHeader
   - Для каждого: 3-5 тест-кейсов

3. **Integration тесты** (API routes):
   - POST /api/tasks — создание (успех + ошибка валидации)
   - PUT /api/tasks/:id — обновление (успех + 404 + 403)
   - Используй mock для Prisma

4. **Покрытие**: какой % покрытия достаточно для Junior проекта?
   (80% для критических путей)

5. Напиши package.json scripts:
   ```
   "test": "jest",
   "test:watch": "jest --watch",
   "test:coverage": "jest --coverage"
   ```

6. Настройка jest.config.js для Next.js.
