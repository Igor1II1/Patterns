## Задание

1. Напиши тесты для утилитарных функций:
   ```
   // lib/utils.ts
   formatDate(date: Date): string
   slugify(text: string): string
   truncate(text: string, maxLength: number): string
   ```

2. Для каждой функции напиши минимум 3 теста:
   - Нормальный случай
   - Граничный случай (пустая строка, null)
   - Некорректный ввод

3. Используй структуру:
   ```
   describe("formatDate", () => {
     it("форматирует дату правильно", () => {
       expect(formatDate(new Date("2026-01-15"))).toBe("15.01.2026")
     })
   })
   ```

4. Объясни: что тестируют unit-тесты? Чем они отличаются от integration?
