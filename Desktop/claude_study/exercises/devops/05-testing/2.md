## Задание

1. Напиши тесты для компонента Button:
   - Рендерится с правильным текстом
   - Вызывает onClick при клике
   - Отключается при disabled
   - Применяет правильный className для variant

2. Напиши тесты для компонента TaskCard:
   - Показывает title и description
   - Показывает Badge с приоритетом
   - Кнопка "Удалить" вызывает onDelete

3. Напиши тесты для формы CreateTaskForm:
   - Рендерит все поля
   - Показывает ошибку при пустом title
   - Вызывает onSubmit с правильными данными
   - Очищает форму после submit

4. Используй: render(), screen.getByText(), userEvent.click(), expect()

5. Объясни: почему тестировать поведение, а не реализацию?
