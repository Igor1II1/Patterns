## Задание

1. Чеклист GitHub-профиля:
   - [ ] Нормальное фото
   - [ ] Bio с позицией и стеком
   - [ ] 3-6 pinned repositories
   - [ ] README профиля (username/username repo)
   - [ ] Регулярные коммиты (зелёные квадратики)
   - [ ] Нет пустых репозиториев и "test123"

2. Напиши README для GitHub-профиля:
   - Привет, кто ты
   - Стек технологий
   - Таблица проектов (название, описание, стек)
   - Контакты

3. Напиши резюме (1 страница):
   - Контакты + GitHub
   - Навыки (по категориям)
   - Проекты (2-3 с конкретикой: что сделал, стек)
   - Образование
   - О себе (1-2 предложения)

4. Правила: что НЕ писать в резюме?
   (Воду, навыки которые не можешь подтвердить, все технологии подряд)
