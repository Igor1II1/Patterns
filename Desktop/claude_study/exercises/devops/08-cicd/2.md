## Задание

1. Напиши CI workflow с PostgreSQL:
   - services.postgres (image, env, ports, healthcheck)
   - DATABASE_URL в env job
   - prisma generate + migrate deploy перед тестами

2. Добавь кеширование:
   - node_modules (по hash package-lock.json)
   - .next/cache (по hash package-lock.json)

3. Напиши CD workflow:
   - needs: test (деплой только после тестов)
   - Деплой на Vercel через vercel-action
   - Секреты: VERCEL_TOKEN, VERCEL_ORG_ID, VERCEL_PROJECT_ID

4. Напиши PR check workflow:
   - Только для pull_request
   - Lint + test + build
   - Комментарий к PR при ошибке

5. Параллельные jobs: lint и test одновременно, build после обоих.
