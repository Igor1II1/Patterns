## Задание

Создай полный CI/CD pipeline для TaskBoard:

1. **.github/workflows/ci.yml** — при push и PR:
   - Lint (параллельно)
   - Type check (параллельно)
   - Tests с PostgreSQL (параллельно)
   - Build (после всех трёх)

2. **.github/workflows/deploy.yml** — при push в main:
   - Запуск CI
   - Деплой на Vercel (после успешного CI)
   - Применение миграций на продакшн БД

3. **.github/workflows/pr-check.yml** — при PR:
   - Все проверки CI
   - Комментарий к PR с результатами
   - Preview deploy на Vercel

4. **Секреты**: перечисли все необходимые.

5. **Кеширование**: npm, .next/cache.

6. Нарисуй схему pipeline (текстом):
   push → lint + test + typecheck → build → deploy
