## Задание

1. Спроектируй набор UI-компонентов для TaskBoard:

   - Dialog (модальное окно)
   - Input + Label
   - Avatar
   - Badge (для приоритета/статуса)

2. Для каждого компонента напиши:
   - Props интерфейс (TypeScript)
   - Какие варианты нужны (cva)
   - Пример использования

3. Создай компонент TaskCard с использованием Card, Badge, Avatar, Button.

4. Объясни: зачем разделять компоненты на ui/ и features/?
