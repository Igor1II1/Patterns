## Задание

Спроектируй полную frontend-архитектуру для TaskBoard:

1. **Компоненты** (полный список с props):
   - UI: Button, Card, Dialog, Input, Select, Badge, Avatar, Dropdown
   - Features: BoardList, BoardView, Column, TaskCard, TaskDialog, UserMenu

2. **API-клиент** (lib/api.ts):
   - Базовая функция fetchApi с обработкой ошибок
   - Функции: getBoards, createBoard, getTasks, createTask, updateTask, moveTask

3. **React Query хуки**:
   - useBoards, useBoard, useTasks
   - useCreateBoard, useCreateTask, useUpdateTask, useMoveTask

4. **Состояние**:
   - Что в React Query (серверное состояние)
   - Что в useState (UI состояние: модалки, фильтры)
   - Что в URL (searchParams: фильтры, текущая доска)

5. **Обработка ошибок**: toast уведомления при ошибках API

6. Объясни каждое решение: почему именно так?
