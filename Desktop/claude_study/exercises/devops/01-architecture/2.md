## Задание

Спроектируй REST API для TaskBoard:

1. Напиши список всех эндпоинтов:
   - Boards: CRUD
   - Columns: CRUD (вложенные в board)
   - Tasks: CRUD (вложенные в column)
   - Users: получение, обновление профиля

2. Для каждого эндпоинта укажи:
   - HTTP-метод (GET/POST/PUT/DELETE)
   - URL
   - Тело запроса (если есть)
   - Формат ответа
   - HTTP-код ответа

3. Пример для tasks:
   ```
   GET    /api/boards/:boardId/columns/:columnId/tasks
   POST   /api/boards/:boardId/columns/:columnId/tasks
   PUT    /api/tasks/:taskId
   DELETE /api/tasks/:taskId
   ```

4. Какие эндпоинты требуют авторизации? Какие — проверки владельца?

5. Напиши формат ошибки: `{ error: string, code: string }`
