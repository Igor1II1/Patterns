## Задание

Спроектируй схему базы данных для приложения "Доска задач" (TaskBoard):

1. Нарисуй ER-диаграмму (текстом или на бумаге) для следующих сущностей:
   - **User** — пользователь (id, name, email, role)
   - **Board** — доска (id, title, ownerId)
   - **Column** — колонка (id, title, boardId, order)
   - **Task** — задача (id, title, description, columnId, assigneeId, order)

2. Определи типы связей:
   - User → Board: какая связь?
   - Board → Column: какая связь?
   - Column → Task: какая связь?
   - User → Task (assignee): какая связь?

3. Напиши Prisma schema для этих моделей.

4. Какие поля нужны для сортировки (drag-and-drop порядок)?

5. Объясни: почему order — число, а не строка?
