## Задание

Спроектируй полную архитектуру для приложения "WeatherApp":

1. **Требования:**
   - Поиск города
   - Текущая погода + прогноз на 5 дней
   - Сохранение избранных городов (авторизация)
   - Адаптивный дизайн
   - API: OpenWeather

2. **Задачи:**
   a) Схема БД (User, FavoriteCity)
   b) REST API эндпоинты
   c) Файловая структура проекта
   d) Компонентное дерево
   e) Список страниц и маршрутов
   f) Server vs Client компоненты
   g) Стратегии рендеринга (SSG/SSR/ISR для каждой страницы)
   h) Env variables

3. Для каждого пункта напиши обоснование выбранного решения.

4. Составь план реализации: что делать первым, что вторым, и т.д.
