   ```
   app/
   ```

   ```
   RootLayout
   ├── Header (навигация, аватар)
   ├── BoardPage
   │   ├── BoardHeader

   │   │   │       └── TaskCard
   ```

3. Для каждого компонента определи: Server или Client? Почему?

4. Какие компоненты переиспользуемые (ui/), какие специфичные (features/)?
