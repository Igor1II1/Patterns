## Задание

1. Напиши lib/logger.ts:
   - Уровни: debug, info, warn, error
   - Формат: [timestamp] [LEVEL] message {data}
   - Фильтрация по LOG_LEVEL из env

2. Добавь логирование в API Route:
   - Начало запроса: logger.info("POST /api/tasks", { body })
   - Успех: logger.info("Task created", { taskId, duration })
   - Ошибка: logger.error("Failed to create task", { error, duration })

3. Перечисли: что логировать, а что НЕТ?
   - ДА: запросы, ошибки, важные действия, время выполнения
   - НЕТ: пароли, токены, номера карт

4. Объясни: зачем логировать время выполнения (duration)?
