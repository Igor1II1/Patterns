## Задание

Создай полную систему мониторинга для TaskBoard:

1. **Logger**: lib/logger.ts с уровнями и структурированными данными
2. **Sentry**: настройка client + server, captureException в API routes
3. **Healthcheck**: /api/health — проверка БД, Redis (если есть)
4. **Vercel Analytics**: подключение в layout.tsx
5. **Vercel Speed Insights**: подключение Web Vitals
6. **UptimeRobot**: настройка мониторинга /api/health
7. **Middleware мониторинг**: X-Response-Time header на каждый API запрос

Для каждого инструмента:
- Напиши код настройки
- Объясни что он отслеживает
- Укажи бесплатный план (лимиты)

Метрики для отслеживания:
- LCP < 2.5 сек, FID < 100 мс, CLS < 0.1
- API Response Time < 200 мс
- Error Rate < 1%
- Uptime > 99.9%
