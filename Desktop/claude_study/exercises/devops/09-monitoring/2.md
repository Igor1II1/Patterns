## Задание

1. Настройка Sentry:
   - Команда установки: npx @sentry/wizard@latest -i nextjs
   - sentry.client.config.ts — DSN, tracesSampleRate
   - sentry.server.config.ts — DSN, enabled только в production

2. Ручной отлов ошибок:
   ```typescript
   Sentry.captureException(error, {
     tags: { endpoint: "/api/tasks" },
     extra: { requestBody: body },
     user: { email: session?.user?.email }
   })
   ```

3. Healthcheck эндпоинт (app/api/health/route.ts):
   - Проверка БД: `await prisma.$queryRaw\`SELECT 1\``
   - Ответ: { status: "ok"/"degraded", database: "connected"/"disconnected" }
   - HTTP-код: 200 или 503

4. Использование healthcheck в Docker Compose.

5. Внешний мониторинг: настройка UptimeRobot (описать шаги).
