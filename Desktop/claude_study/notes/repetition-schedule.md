# Расписание повторений (Spaced Repetition)

> Этот файл читается Claude в начале каждой сессии.
> Claude автоматически подбирает прогрев на основе расписания.

---

## Как работает система

Каждая тема повторяется через увеличивающиеся интервалы:
```
Изучил → 1 день → 3 дня → 7 дней → 14 дней → 30 дней → ЗАКРЕПЛЕНО
```

Если при повторении были ошибки → интервал сбрасывается:
- 1-2 ошибки → назад на 1 шаг (например 14д → 7д)
- 3+ ошибок → назад к 1 дню

**Типы повторений:**
- **К** = карточка-минимум (2-3 мин, одна концепция)
- **З** = задание из микса (5-10 мин)
- **П** = полный практикум (вся сессия)

## Правила обновления

1. Claude обновляет этот файл в **конце каждой сессии**
2. Если тема повторена успешно → ставим ✅, переходим к следующему интервалу
3. Если были ошибки → сбрасываем интервал, записываем новую дату
4. Новая тема → добавляем строку с датой изучения
5. В начале сессии → Claude читает "Что повторять СЕГОДНЯ" и строит прогрев
