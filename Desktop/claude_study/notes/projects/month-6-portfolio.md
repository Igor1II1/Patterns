## Выбор проекта

Выбери ONE из вариантов — тот что интереснее:

### Вариант A: Менеджер задач с командами
- Создавать задачи, присваивать участников
- Статусы: Backlog → In Progress → Review → Done
- Drag-and-drop между колонками
- Комментарии к задачам
- Уведомления (через веб-сокеты или polling)

### Вариант B: Платформа рецептов
- Публикация рецептов с пошаговыми инструкциями
- Поиск по ингредиентам, кухне, времени приготовления
- Список покупок (авто-генерация из рецепта)
- Рейтинг и избранное

### Вариант C: Трекер привычек
- Создание привычек с целями (ежедневно / еженедельно)
- Статистика выполнения за период
- Streak (серия без пропусков)
- Графики прогресса

### Вариант D: Свой проект
Если есть идея — обсуди с Claude, и вместе оценим реалистичность за 2 месяца.

## Обязательные технические требования

```
Frontend:   Next.js 14 + TypeScript + Tailwind + Shadcn/ui
Backend:    Next.js API Routes или отдельный Express/Nest
Database:   PostgreSQL + Prisma
Auth:       NextAuth.js (минимум 1 провайдер: GitHub / Google)
Deploy:     Vercel (frontend) + Railway (database)
Код:        Публичный GitHub репозиторий
README:     Полный README со скриншотами и инструкцией запуска
```
