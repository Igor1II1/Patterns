## Как читать статистику

**Формула расчёта уровня владения:**

```
Уровень = (Теория_% × 0.4) + (Практика_% × 0.4) + (Самостоятельность × 0.2)

Теория_%     = (✅ × 1.0 + ⚠️ × 0.5 + 💬 × 0.3) / всего_вопросов × 100
Практика_%   = (✅ × 1.0 + ⚠️ × 0.5) / всего_заданий × 100
Самостоятельность = экспертная оценка 0–100%:
  100% — решил всё сам, без подсказок
   70% — 1-2 подсказки, но идею понимал
   40% — нужны были наводящие вопросы
   10% — не мог начать без помощи
```

**Пример:** Теория 60% × 0.4 + Практика 85% × 0.4 + Самостоятельность 50% × 0.2 = 24 + 34 + 10 = **68% 🟡**

**Шкала уровня:**
- 🔴 `0-40%` — тема знакома поверхностно, нужно повторить
- 🟡 `40-70%` — базовое понимание, есть пробелы
- 🟢 `70-90%` — уверенное знание, редкие ошибки
- ⭐ `90-100%` — отличное знание, готов к собеседованию

**Статусы заданий:**
- ✅ Выполнено без ошибок
- ⚠️ Выполнено с ошибками (описаны ниже)
- ❌ Не выполнено / не пробовал
- 💬 Был вопрос к Claude

**Система повторений (5 слоёв):**
- Слой 1: Ежедневный прогрев — карточки по расписанию из `repetition-schedule.md` (+1д, +3д, +7д, +14д, +30д)
- Слой 2: Вплетение — каждое задание использует 1-2 концепции из прошлых тем
- Слой 3: Чередование — практика = 70% новое + 30% старое
- Слой 4: Еженедельный микс — 5-7 заданий из разных тем (последний день недели)
- Слой 5: Практикум — каждые 3 темы, 100% покрытие всех пройденных концепций
