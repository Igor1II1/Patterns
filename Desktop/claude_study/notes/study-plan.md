# План обучения — ежедневная программа

> Обновляется Claude в начале каждого месяца.
> Читается автоматически в начале каждой сессии вместе с другими файлами.

---

## Базовая математика

```
Доступно:       3-4 ч/день × 6 дней/нед × 26 нед = ~546 часов
Нужно:          ~250 часов контента
Запас:          ~300 часов на повторение, сложные темы, проекты
Темп:           комфортный — можно углубляться, не торопиться
```

---

## Структура каждой сессии (3-4 часа)

```
[10 мин]  Прогрев       — 2-3 вопроса по теме ПРОШЛОЙ сессии
[40 мин]  Теория        — читаешь главу учебника самостоятельно
[20 мин]  Опрос         — Claude задаёт вопросы по новой теме
[90 мин]  Практика      — задания от 🟢 к 🔴, пишешь код сам
[20 мин]  Разбор        — Claude проверяет, объясняет ошибки
[10 мин]  Итог          — обновление файлов, план на завтра
```

**Правило:** если тема не усвоена — не переходи к следующей.
**Правило:** каждую сессию начинай с прогрева по прошлой теме.

## Как Claude использует этот файл

В начале каждой сессии:
1. Читает этот файл + `progress.md` + `time-tracker.md`
2. Смотрит — идём по плану или отстаём?
3. Говорит Игорю что делаем СЕГОДНЯ конкретно (тема + задания)
4. Если тема не усвоена — оставляем ещё день, сдвигаем план

В конце месяца:
- Обновляет план следующего месяца с учётом реального прогресса
