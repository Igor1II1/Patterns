**🟡 Б1.11.3** Создай простой Proxy для валидации:

```js
function createValidatedUser(data) {
  return new Proxy(data, {
    set(target, prop, value) {
      // Правила:
      // - name: только строка, не пустая
      // - age: только число от 0 до 150
      // - остальные поля: без ограничений
      // Бросай TypeError при нарушении
    }
  });
}

const user = createValidatedUser({ name: "Игорь", age: 25 });
user.name = "Анна";     // ✅
user.age = 30;          // ✅
user.age = -1;          // ❌ TypeError: Возраст должен быть от 0 до 150
user.name = "";         // ❌ TypeError: Имя не может быть пустым
user.name = 42;         // ❌ TypeError: Имя должно быть строкой
user.city = "Киев";     // ✅ (нет ограничений)
```
