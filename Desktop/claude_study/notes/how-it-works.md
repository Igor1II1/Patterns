## Как начать новую сессию

1. Открываешь Claude Code в папке `claude_study/`
2. `CLAUDE.md` читается **автоматически** -- Claude уже знает инструкции
3. Пишешь: **"начинаем, 11:25"** (с текущим временем)
4. Claude читает 10 файлов и говорит:
   - блок статистики (прогресс, часы, отставание/опережение)
   - что проходили в прошлый раз
   - что слабо и нужно повторить
   - что делаем сегодня

> Если время не указал -- Claude спросит сам.

---

## Как проходит занятие

```
1. Прогрев     -> Claude задаёт 2-3 вопроса по ПРОШЛОЙ теме (5 мин)
2. Теория      -> читаешь главу учебника (js/*.md)
3. Опрос       -> Claude задаёт все вопросы по НОВОЙ теме
                  ты отвечаешь как на собеседовании -> идёт в статистику
4. Практика    -> делаешь задания из exercises.md (от простых к сложным)
5. Проверка    -> пишешь "готово", Claude проверяет код
6. Итоги       -> пишешь "закрываем, 17:00", Claude обновляет 8 файлов
```

## График теоретических проверок

| Когда | Что | Вопросов |
|-------|-----|---------|
| Начало каждой сессии | Прогрев по прошлой теме | 2-3 |
| После новой темы | Полный опрос по теме | Все вопросы темы |
| Раз в 2 недели | Сводный опрос | 10-15 |
| Конец Блока 0 | Финальный экзамен | Все 60 |

Это называется **интервальное повторение** -- самый эффективный способ запомнить надолго.

## Статистика -- как читать

В `progress.md` уровень владения темой:

| Значок | Уровень | Что означает |
|--------|---------|-------------|
| 🔴 0-40% | Слабо | Нужно повторить |
| 🟡 40-70% | Базово | Есть пробелы |
| 🟢 70-90% | Уверенно | Редкие ошибки |
| ⭐ 90-100% | Отлично | Готов к собеседованию |

## Задания -- как делать

1. Claude выдаёт задание по формату (правило 6 из improvement-requests.md)
2. Каждое задание содержит: ссылки на учебник + полное описание + тесты
3. Пишешь код в `exercises/block-X/XX-тема/script.js`
4. Говоришь **"готово"** -- Claude проверяет
