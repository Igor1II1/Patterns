## Ключевые концепции

### Инкапсуляция
**Идея:** объединение данных и методов в одном объекте + контроль доступа.

Два аспекта:
- **Группировка** — данные и логика живут вместе
- **Сокрытие** — внутренние детали скрыты, наружу только публичные методы

Приватные поля (`#field`) — это **инструмент** для реализации инкапсуляции, а не сама инкапсуляция.

```js
class BankAccount {
  #balance = 0;  // приватное — нельзя изменить напрямую

  deposit(amount) {
    if (amount <= 0) throw new Error("Сумма должна быть положительной");
    this.#balance += amount;
  }

  getBalance() {
    return this.#balance;
  }
}
```

### Наследование
**Идея:** дочерний класс получает свойства и методы родителя. Не нужно писать один и тот же код дважды.

```js
class Book {
  constructor(title, author) {
    this.title = title;
    this.author = author;
  }
  getInfo() {
    return `${this.title}, ${this.author}`;
  }
}

class Ebook extends Book {
  constructor(title, author, format) {
    super(title, author);  // вызов конструктора родителя
    this.format = format;
  }
}
```

**`super()`** — обязателен в конструкторе наследника. Вызывает конструктор родителя, чтобы инициализировать унаследованные свойства. Без `super()` будет ошибка.

### Полиморфизм
**Идея:** один и тот же метод работает по-разному в зависимости от объекта. Вызывающему коду всё равно, какой конкретно объект — он вызывает один метод, а результат разный.

```js
class Animal {
  speak() { return "..."; }
}
class Dog extends Animal {
  speak() { return "Гав!"; }
}
class Cat extends Animal {
  speak() { return "Мяу!"; }
}

// Полиморфизм в действии:
[new Dog(), new Cat()].forEach(a => console.log(a.speak()));
```

Переопределение метода — **инструмент**, через который полиморфизм реализуется.

### Абстракция
**Идея:** скрытие сложности, предоставление простого интерфейса.

JS не имеет нативных абстрактных классов (как в Java/C#). Но можно имитировать через `new.target`:

```js
class Publication {
  constructor() {
    if (new.target === Publication) {
      throw new Error("Publication нельзя создать напрямую!");
    }
  }
}
```

**`new.target`** — показывает, какой класс вызвали через `new`. Формула: **new.target = класс после new**.
