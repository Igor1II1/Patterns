# Трекер слабых мест

> Этот файл читается Claude в начале каждой сессии.
> Каждая ошибка/слабость отслеживается до закрытия.
> Закрытие = правильный ответ/решение БЕЗ подсказок.

---

## Как работает

- **ОТКРЫТО** — ошибка зафиксирована, ещё не проверена повторно
- **ЗАКРЫТО** — повторил и ответил/сделал правильно без подсказок (с датой)
- **ПОВТОРИЛАСЬ** — пробовал закрыть, но снова ошибся (обнуляем)

**Правила:**
1. Новая ошибка → добавить строку со статусом ОТКРЫТО
2. На каждом прогреве → 1-2 задания/вопроса из ОТКРЫТЫХ (приоритет: старые + повторяющиеся)
3. Правильно ответил без подсказки → ЗАКРЫТО + дата
4. Снова ошибся → ПОВТОРИЛАСЬ + дата (остаётся ОТКРЫТО, приоритет повышается)
5. Если ошибка повторялась 3+ раз → добавить в прогрев как обязательную карточку
