## JavaScript

| Термин | Определение | Глава |
|--------|-------------|-------|
| **Переменная** | Именованный контейнер для хранения значения | [[js/01-variables]] |
| **Тип данных** | Вид значения: `string`, `number`, `boolean`, `null`, `undefined`, `object`, `symbol` | [[js/01-variables]] |
| **Примитив** | Неизменяемое простое значение (строка, число, булево, null, undefined) | [[js/01-variables]] |
| **Объект** | Изменяемая структура данных из пар ключ-значение | [[js/07-objects]] |
| **Ссылочный тип** | Переменная хранит ссылку на объект в памяти, не само значение | [[js/07-objects]] |
| **Функция** | Блок кода, который можно вызвать по имени | [[js/04-functions]] |
| **Коллбэк** | Функция, переданная другой функции как аргумент | [[js/04-functions]] |
| **Стрелочная функция** | Сокращённый синтаксис функции: `() => {}` | [[js/04-functions]] |
| **Рекурсия** | Функция, вызывающая саму себя | [[js/04-functions]] |
| **Замыкание** | Функция + её лексическое окружение (переменные из внешнего скопа) | [[js-advanced/02-closures]] |
| **Область видимости** | Зона кода, в которой доступна переменная | [[js-advanced/02-closures]] |
| **Лексическая область** | Область видимости определяется местом написания кода, а не вызова | [[js-advanced/02-closures]] |
| **IIFE** | Immediately Invoked Function Expression — функция, вызванная сразу после объявления | [[js-advanced/02-closures]] |
| **Прототип** | Объект-предок, от которого наследуются свойства и методы | [[js-advanced/03-prototypes]] |
| **Прототипная цепочка** | Цепочка объектов через `[[Prototype]]` до `null` | [[js-advanced/03-prototypes]] |
| **`this`** | Ссылка на объект контекста вызова | [[js/08-this]] |
| **Деструктуризация** | Распаковка значений из массива/объекта в переменные | [[js/07-objects]] |
| **Spread `...`** | Разворачивание массива/объекта: `[...arr]` | [[js/07-objects]] |
| **Rest `...`** | Сбор оставшихся аргументов в массив: `function f(...args)` | [[js/04-functions]] |
| **Шаблонный литерал** | Строка с интерполяцией: `` `Привет, ${name}` `` | [[js/05-strings]] |
| **Цикл (loop)** | Конструкция для повторения блока кода | [[js/03-loops]] |
| **for** | Цикл со счётчиком: `for (let i = 0; i < n; i++)` | [[js/03-loops]] |
| **while** | Цикл с условием: выполняется пока условие true | [[js/03-loops]] |
| **do...while** | Как while, но тело выполнится хотя бы один раз | [[js/03-loops]] |
| **for...of** | Перебор значений итерируемого объекта (массив, строка) | [[js/03-loops]] |
| **for...in** | Перебор ключей объекта (строки). Не для массивов! | [[js/03-loops]] |
| **break** | Досрочный выход из цикла | [[js/03-loops]] |
| **continue** | Пропуск текущей итерации цикла | [[js/03-loops]] |
| **Итерация** | Один проход тела цикла; перебор элементов коллекции | [[js/03-loops]] |
| **Итератор** | Объект с методом `next()`, возвращающим `{value, done}` | [[js-advanced/09-iterators]] |
| **Генератор** | Функция с `yield`, которая может приостанавливаться | [[js-advanced/09-iterators]] |
| **Map** | Коллекция пар ключ-значение. Ключ любого типа | [[js-advanced/06-map-set]] |
| **Set** | Коллекция уникальных значений | [[js-advanced/06-map-set]] |
| **JSON** | Текстовый формат данных для передачи и хранения объектов | [[js-advanced/05-json]] |
| **RegExp** | Регулярное выражение — паттерн для поиска/замены в строках | [[js-advanced/07-regexp]] |
